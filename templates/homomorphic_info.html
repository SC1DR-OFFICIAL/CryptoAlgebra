Гомоморфное шифрование и цифровая подпись

В разработанной системе электронного голосования применяется аддитивно-гомоморфная схема шифрования Пэйе (Paillier). Она позволяет складывать зашифрованные голоса без их предварительной расшифровки. Это гарантирует конфиденциальность выбора: сервер не имеет доступа к содержимому бюллетеней, но может корректно посчитать общее количество голосов.

Каждый пользователь на стороне клиента шифрует свой голос с использованием открытого ключа схемы Paillier. После этого формируется вектор, в котором единица зашифрована только на позиции выбранного варианта ответа.

Для подтверждения подлинности каждый зашифрованный голос дополнительно подписывается цифровой подписью RSA, которая создаётся с использованием приватного ключа пользователя. Подпись позволяет убедиться, что голос подан зарегистрированным участником и не был подделан.

При подсчёте все зашифрованные векторы агрегируются с помощью гомоморфного сложения, и лишь итоговый результат расшифровывается с использованием закрытого ключа сервера. Таким образом, обеспечивается анонимный, проверяемый и математически корректный подсчёт.
